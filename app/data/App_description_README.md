## What The App Does

This quiz app lets users pick a category and answer multiple-choice questions under a 15-second timer per question. It tracks the score, gives instant feedback after each answer, and shows a final result summary with performance insights.

## Core Functionality

1. Category selection from four topics: Technology, Science, History, and Math.
2. Timed question flow with automatic timeout handling.
3. Immediate visual feedback for correct, wrong, and timed-out answers.
4. Progress tracking using a question counter and progress bar.
5. Final results screen with score percentage, summary message, and quick return to home.

## UI Structure

1. Header area with branding and theme toggle.
2. Home screen with a gradient hero section and category cards.
3. Quiz screen with timer badge, question card, answer option cards, and action button.
4. Results screen with gradient summary card, circular score visualization, and stat cards.
5. Consistent spacing, rounded cards, and icon-based affordances for clear navigation.
