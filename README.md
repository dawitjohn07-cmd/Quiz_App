# Quiz App 📱

A mobile quiz application built with React Native and Expo that allows users to test their knowledge across different categories with timed multiple-choice questions, instant feedback, and performance tracking.

## Features

- 📚 Multiple quiz categories:
  - Technology
  - Science
  - History
  - Math

- ⏱️ Timed quiz experience with a 15-second limit per question
- ✅ Instant answer feedback with correct/wrong indicators
- 📊 Progress tracking and question counter
- 🏆 Final score summary with performance evaluation
- 🌙 Theme support with consistent UI styling
- 📱 Responsive mobile-first design
